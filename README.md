## Sudoku UI

Static Sudoku puzzle **UI** (no backend) with:

- **9×9 grid** with 3×3 subgrid borders
- **Click/tap to select** cells
- **Keyboard input** (1–9), arrow keys to move, Delete/Backspace to erase
- **Notes mode** (toggle with **N**)
- **Conflict highlighting**
- **New puzzle / Reset / Check**

### Run it

Option A (simplest): open `index.html` directly in your browser.

Option B (recommended): run a small local server (better for some browsers).

```bash
cd C:\Users\darim_xn8qmj8\Code\sudoku-ui
python -m http.server 5173
```

Then open `http://localhost:5173`.
